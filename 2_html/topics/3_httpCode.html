Http常见状态码（简洁易懂）- 前端念初 - csdn HTTP状态码 - 百度百科

状态码：客服端发出http请求，服务器或者缓存返回的一个标志，方便分类定位具体错误；

1xx 信息状态码

代表请求已被接受，需要继续处理。这类响应是临时响应，一般不做特殊设置，服务器是不会返回的，
所以也不常见；
100 Continue
通知客户端它的部分请求已经被服务器接收，且仍未被拒绝。客户端应当继续发送请求的剩余部分；

2xx 成功状态码

服务请求成功，
200 ok
201 Created
202 Accepted

3xx 重定向状态码

客户端采取进一步的操作才能完成请求。
301: 永久重定向
302：临时重定向, 没权限时后端重定向
304 Not Modified
    协商缓存起效果：last Modified / etag起效果时

4xx 客户端错误码

400 Bad Request：
    服务器端无法理解客户端发送的请求，请求报文中可能存在语法错误。
403 Forbidden: 没权限
404 Not Found:

5xx 服务器错误码

500 Internal Server Error
502 Bad Gateway
    作为网关或者代理工作的服务器尝试执行请求时，从上游服务器接收到无效的响应。
503 Service Unavailable：
抱歉，我现在正在忙着。该状态码表明服务器暂时处于超负载或正在停机维护，现在无法处理请求。
